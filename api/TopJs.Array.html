将数组转换成一个键值对常量对象，键由数组值或者指定的函数得到，值是相关值在数组的索引值+1

 let map = TopJs.Array.toMap(['a','b','c']);

 // map = { a: 1, b: 2, c: 3 };
 //或者key的值可以通过数组中元素指定

 let map = TopJs.Array.toMap([
             { name: 'a' },
             { name: 'b' },
             { name: 'c' }
         ], 'name');

 //我们也能指定一个key获取函数

             { name: 'a' },
             { name: 'b' },
             { name: 'c' }

(static) toValueMap(array, getKeyopt, scopeopt, arrayify) → {Object}

Source:
  • lang/Array.js, line 718

将一个数组转换成一个由其值作为键值或者指定的getKey生成其键值，然后由数组当前项 作为值得常量对象


var map = TopJs.Array.toValueMap(['a','b','c']);
// map = { a: 'a', b: 'b', c: 'c' };

//直接指定键名称
var map = TopJs.Array.toValueMap([
             { name: 'a' },
             { name: 'b' },
             { name: 'c' }
         ], 'name');

// map = { a: {name: 'a'}, b: {name: 'b'}, c: {name: 'c'} };

// 制定一个生成键名称的函数
var map = TopJs.Array.toValueMap([
             { name: 'a' },
             { name: 'b' },
             { name: 'c' }
         ], function (obj) { return obj.name.toUpperCase(); });

// map = { A: {name: 'a'}, B: {name: 'b'}, C: {name: 'c'} };
Parameters:

指定当键已经存在于结果对象中时候如何处理值
arrayify = 1表示强制将所有相同键的值生成数组
arrayify = 2表示如果存在的键当前的值是数组那么将当前值添加到数组，否则直接覆盖
其他, 表示如果存在相同的键就直接覆盖之前的数据
