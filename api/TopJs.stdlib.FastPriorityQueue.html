FastPriorityQueue

TopJs.stdlib. FastPriorityQueue

This is an efficient implementation of an integer priority queue in PHP

This class acts like a queue with TopJs.stdlib.FastPriorityQueue#insert and TopJs.stdlib.FastPriorityQueue#extract, removing the elements from the queue and it also acts like an Iterator without removing the elements. This behaviour can be used in mixed scenarios with high performance boost.
